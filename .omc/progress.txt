**Key patterns learned:**
- FastAPI lifespan context manager for catalog warmup (replaces deprecated @app.on_event)
- google-auth + googleapiclient preferred over PyDrive2 for service account auth
- ZIP_STORED is correct for bundles of already-compressed files (.zip, .pww)
- threading.Lock() at module level is safe for FastAPI (sync catalog building)
- ES modules avoid need for bundler; import maps would be next step if deps needed
- Catalog degrades gracefully to empty lists (not exception) on Drive API failure
